# ChatBot

https://expo.dev/@abhirup/chatbot-expo

## Tech Stack

**Client:** Expo, React-Native, React

## Run Locally

Clone the project

```bash
  git clone https://github.com/abhirupkumar/Chatbot-expo.git
```

Go to the project directory

```bash
  cd Chatbot-expo
```

Install dependencies

```bash
  npm install
```
